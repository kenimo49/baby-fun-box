# Balloon Pop - バルーンポップ

## 概要

画面に浮かぶカラフルな風船をタップ（クリック）すると、「ポン！」と弾けてキラキラが飛び散る、1〜2歳児向けのシンプルなゲームです。

## 対象年齢

**1〜2歳**

### この年齢に適している理由

- **大きなタッチターゲット**: 風船は半径60〜100ピクセルと大きめ
- **シンプルな操作**: タップ/クリックのみ
- **即座のフィードバック**: 弾けるアニメーション + 効果音
- **鮮やかな色彩**: 7色のカラフルな風船
- **繰り返しの楽しさ**: 風船は次々と出現

## 遊び方

1. 画面下から風船がゆっくり浮かび上がってきます
2. 風船をクリック（タップ）すると弾けます
3. 弾けるとキラキラのパーティクルが飛び散ります
4. 新しい風船が次々と出現します

## 操作方法

| 操作 | アクション |
|------|----------|
| クリック/タップ | 風船を弾かせる |
| 戻るボタン（左上） | ランチャーに戻る |
| ESC キー | ランチャーに戻る |

## 実行方法

### 統合アプリ経由

```bash
# プロジェクトルートから
python main.py
# ランチャーでBalloon Popを選択
```

### 単体実行（開発用）

```bash
# プロジェクトルートから
python apps/balloon_pop/main.py
```

## 技術的な詳細

### クラス構成

```
BalloonPopGame (BaseGame)  - ゲーム全体の管理
├── Balloon                - 風船オブジェクト（位置、色、動き）
├── Particle               - 弾けたときのパーティクル効果
└── BackButton             - ランチャーに戻るボタン
```

### 特徴

- **BaseGame継承**: ランチャーからの統一的な呼び出しに対応
- **プロシージャル効果音**: 外部ファイル不要で「ポン」音を生成
- **パーティクルシステム**: 弾けたときの視覚的フィードバック
- **ゆらゆらアニメーション**: 風船が自然に揺れながら上昇
