# おえかきらくがき カスタマイズガイド

## 概要

スタンプ画像や効果音をカスタマイズできます。
ファイルを配置するだけで自動的に読み込まれます。

## ディレクトリ構造

```
apps/oekaki_rakugaki/
├── assets/
│   ├── images/     # スタンプ画像
│   │   ├── star.png
│   │   ├── heart.png
│   │   └── flower.png
│   └── sounds/     # 効果音
│       ├── pop.wav
│       └── sparkle.wav
└── game.py
```

## スタンプ画像のカスタマイズ

### 対応フォーマット

- **PNG** (推奨、透過対応)
- **JPG**
- **BMP**

### ファイル名規則

| スタンプ | ファイル名 |
|----------|-----------|
| 星 | `star.png` |
| ハート | `heart.png` |
| 花 | `flower.png` |

### 推奨サイズ

- **80x80 px** 推奨
- 透過 PNG で作成すると綺麗に表示されます

### 画像作成のコツ

1. **背景は透過**にする（PNG形式）
2. **シンプルな形状**を心がける
3. **明るく鮮やかな色**を使用
4. **中心に配置**する

## 効果音のカスタマイズ

### 対応フォーマット

- **WAV** (推奨)
- **OGG**
- **MP3**

### ファイル名規則

| 音 | ファイル名 | 用途 |
|----|-----------|------|
| ポップ音 | `pop.wav` | 描画開始・ボタンタップ時 |
| キラキラ音 | `sparkle.wav` | スタンプ配置・クリア時 |

### 推奨仕様

| 項目 | 推奨値 |
|------|--------|
| サンプルレート | 22050〜44100 Hz |
| ビット深度 | 16-bit |
| チャンネル | モノラル |
| 長さ | 0.05〜0.2秒 |

### 音声作成のコツ

1. **短い音**が適切（0.1秒程度）
2. **明るい音**を選ぶ
3. **急な音量変化**を避ける

## 自動読み込み機能

カスタムアセットは**ゲーム起動時に自動的に読み込まれます**。

1. ファイルを `assets/images/` または `assets/sounds/` に配置
2. ゲームを起動
3. 自動的にカスタムアセットが使用される

カスタムアセットが見つからない場合は、デフォルトのプリミティブ描画・手続き的音声生成が使用されます。

### 読み込み優先順位

**画像**: `png` → `jpg` → `bmp`

**音声**: `wav` → `ogg` → `mp3`

## フリー素材リソース

### 画像素材

- [いらすとや](https://www.irasutoya.com/)
- [イラストAC](https://www.ac-illust.com/)

### 音声素材

- [効果音ラボ](https://soundeffect-lab.info/)
- [Freesound](https://freesound.org/)

## 注意事項

1. **ライセンスを確認**してから素材を使用
2. **ファイルサイズ**は小さく（画像は 100KB 以下推奨）
3. **ファイル名**は正確に（大文字小文字に注意）
