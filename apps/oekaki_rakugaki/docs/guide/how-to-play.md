# おえかきらくがき の遊び方

## 基本的な遊び方

### 1. ゲームを起動

ランチャーから「Oekaki」を選択、または単体で起動：

```bash
python apps/oekaki_rakugaki/main.py
```

### 2. 画面の見方

```
┌─────────────────────────────────────────────────────┐
│ [←]  おえかきらくがき                    [クリア]   │ ← ヘッダー
├─────────────────────────────────────────────────────┤
│                                                     │
│                                                     │
│                  キャンバス領域                      │ ← ここにお絵かき
│                  (白い背景)                         │
│                                                     │
│                                                     │
├─────────────────────────────────────────────────────┤
│  [🔴][🟡][🟢][🔵][🟣][🟠][🌸] │ [●][◉][⬤] │ [⭐❤🌸]│
│        色パレット              │  サイズ   │ スタンプ│
└─────────────────────────────────────────────────────┘
```

### 3. お絵かきする

キャンバス（白い領域）を指でなぞると、線が描けます！

1. **タッチして動かす** → 線が描ける
2. **指を離す** → 描き終わり

### 4. 色を変える

画面下部の色ボタンをタップすると、ペンの色が変わります：

| 色 | 名前 |
|----|------|
| 🔴 | あか |
| 🟡 | きいろ |
| 🟢 | みどり |
| 🔵 | あお |
| 🟣 | むらさき |
| 🟠 | オレンジ |
| 🌸 | ピンク |

### 5. ペンの太さを変える

サイズボタンで太さを選べます：

| ボタン | 太さ |
|--------|------|
| ● | ほそい |
| ◉ | ふつう |
| ⬤ | ふとい |

### 6. スタンプを使う

スタンプボタンをタップすると、スタンプモードに切り替わります：

| スタンプ | 名前 |
|----------|------|
| ⭐ | ほし |
| ❤️ | ハート |
| 🌸 | はな |

スタンプモードでキャンバスをタップすると、その場所にスタンプが配置されます。

### 7. 消す

「クリア」ボタンをタップすると、キャンバスが真っ白に戻ります。

## 操作方法

| 操作 | 動作 |
|------|------|
| キャンバスをドラッグ | 線を描く |
| キャンバスをタップ（スタンプモード）| スタンプを配置 |
| 色ボタンをタップ | 色を変更 |
| サイズボタンをタップ | ペンの太さを変更 |
| スタンプボタンをタップ | スタンプモードに切り替え |
| クリアボタン | キャンバスをリセット |
| 戻るボタン [←] | ランチャーに戻る |
| ESC キー | ランチャーに戻る |

## お子さまへの声かけ例

- 「どんな色でかこうかな〜？」
- 「あ！赤い線がかけたね！」
- 「お星さまをペタン！きれいだね〜」
- 「太い線でぐるぐる〜」

## トラブルシューティング

### 線が描けない

- キャンバス（白い領域）の中でタッチしているか確認
- スタンプモードになっていないか確認（色ボタンをタップすると通常モードに戻ります）

### スタンプが置けない

- スタンプボタンをタップしてスタンプモードに切り替えてください
- キャンバス内でタップしてください

### 音が出ない

- システムの音量を確認してください
