# もぐらたたき カスタマイズガイド

## 概要

キャラクター画像や効果音をカスタマイズできます。
ファイルを配置するだけで自動的に読み込まれます。

## ディレクトリ構造

```
apps/mogura_tataki/
├── assets/
│   ├── images/     # キャラクター画像
│   │   ├── rabbit.png
│   │   ├── bear.png
│   │   ├── cat.png
│   │   ├── dog.png
│   │   ├── chick.png
│   │   └── frog.png
│   └── sounds/     # 効果音
│       ├── pop.wav
│       └── tap.wav
└── game.py
```

## キャラクター画像のカスタマイズ

### 対応フォーマット

- **PNG** (推奨、透過対応)
- **JPG**
- **BMP**

### ファイル名規則

| キャラクター | ファイル名 |
|-------------|-----------|
| うさぎ | `rabbit.png` |
| くま | `bear.png` |
| ねこ | `cat.png` |
| いぬ | `dog.png` |
| ひよこ | `chick.png` |
| かえる | `frog.png` |

### 推奨サイズ

- **100x100 px** 推奨
- 正方形で作成
- 透過 PNG で作成すると綺麗に表示

### 画像作成のコツ

1. **背景は透過**にする
2. **顔を大きく**描く（穴から顔だけ出る）
3. **シンプルな形状**を心がける
4. **明るい色**を使用

## 効果音のカスタマイズ

### 対応フォーマット

- **WAV** (推奨)
- **OGG**
- **MP3**

### ファイル名規則

| 音 | ファイル名 | 用途 |
|----|-----------|------|
| ポップ音 | `pop.wav` | 動物が出てきた時 |
| タップ音 | `tap.wav` | 動物をタッチした時 |

### 推奨仕様

| 項目 | 推奨値 |
|------|--------|
| サンプルレート | 22050〜44100 Hz |
| ビット深度 | 16-bit |
| チャンネル | モノラル |
| 長さ | 0.1〜0.2秒 |

### 音声作成のコツ

1. **短い音**が適切
2. **明るく楽しい音**を選ぶ
3. **タップ音は達成感のある音**に

## 自動読み込み機能

カスタムアセットは**ゲーム起動時に自動的に読み込まれます**。

1. ファイルを `assets/images/` または `assets/sounds/` に配置
2. ゲームを起動
3. 自動的にカスタムアセットが使用される

カスタムアセットが見つからない場合は、デフォルトのプリミティブ描画・手続き的音声生成が使用されます。

### 読み込み優先順位

**画像**: `png` → `jpg` → `bmp`

**音声**: `wav` → `ogg` → `mp3`
