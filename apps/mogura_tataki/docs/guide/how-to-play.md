# もぐらたたき の遊び方

## 基本的な遊び方

### 1. ゲームを起動

ランチャーから「Mogura」を選択、または単体で起動：

```bash
python apps/mogura_tataki/main.py
```

### 2. スタート画面

```
┌──────────────────────────────────────────────┐
│ [←]                                          │
│                                              │
│            もぐらたたき                       │
│                                              │
│   30びょうで どうぶつを たくさんタッチしよう！  │
│                                              │
│          [ゲームスタート！]                   │
│                                              │
└──────────────────────────────────────────────┘
```

**「ゲームスタート！」** ボタンをタッチしてゲーム開始！

### 3. プレイ画面

```
┌──────────────────────────────────────────────┐
│ [←]      スコア: 5            30             │
├──────────────────────────────────────────────┤
│                                              │
│     ⭕        🐰        ⭕                   │
│    (穴)     (うさぎ)    (穴)                  │
│                                              │
│     🐻        ⭕        🐱                   │
│   (くま)     (穴)      (ねこ)                 │
│                                              │
└──────────────────────────────────────────────┘
```

- **タイマー（右上）**: 残り時間（30秒からカウントダウン）
- **スコア（中央上）**: タッチした動物の数

### 4. 動物が出てくる

穴からかわいい動物が顔を出します：

- 🐰 うさぎ（ピンク）
- 🐻 くま（オレンジ）
- 🐱 ねこ（きいろ）
- 🐶 いぬ（みずいろ）
- 🐤 ひよこ（きいろ）
- 🐸 かえる（みどり）

### 5. タッチする

動物が出てきたら、顔をタッチ！

- タッチすると **音が鳴る**
- **キラキラ** が飛び出す
- 動物は **引っ込む**
- **スコアが +1** される

### 6. 結果画面

30秒経つとゲーム終了！

```
┌──────────────────────────────────────────────┐
│ [←]                                          │
│                                              │
│               けっか                          │
│                                              │
│              15 てん                          │
│                                              │
│             すごーい！                        │
│                                              │
│            [もういちど]                       │
│                                              │
└──────────────────────────────────────────────┘
```

**「もういちど」** ボタンでスタート画面に戻ります。

## 操作方法

| 操作 | 動作 |
|------|------|
| ゲームスタートボタン | ゲーム開始 |
| 動物をタッチ | スコア+1＋エフェクト |
| もういちどボタン | スタート画面に戻る |
| 戻るボタン [←] | ランチャーに戻る |
| ESC キー | ランチャーに戻る |

## ゲームの特徴

### 30秒チャレンジ

- 赤ちゃんの集中力に合わせた短い時間
- 残り10秒になるとタイマーが赤くなる

### ゆっくりペース

- 動物は **2〜4秒** 出ています
- 焦らなくても大丈夫！

### 褒め言葉

結果画面ではスコアに応じた褒め言葉が表示されます：

| スコア | メッセージ |
|--------|----------|
| 15点以上 | すごーい！ |
| 10〜14点 | がんばったね！ |
| 5〜9点 | いいね！ |
| 0〜4点 | たのしかったね！ |

### タッチしなくてもOK

- タッチしなくても動物は自然に引っ込みます
- どんなスコアでも褒め言葉がもらえます

## お子さまへの声かけ例

- 「あ！うさぎさんだ！タッチできるかな？」
- 「くまさんが出てきたね〜」
- 「すごい！タッチできたね！」
- 「次はどこから出てくるかな〜？」

- 「〇〇てん！がんばったね！」
- 「もう一回やる？」

## トラブルシューティング

### 動物が出てこない

- ゲームスタートボタンを押しましたか？
- しばらく待ってみてください（最初は1秒後に出現）
- ゲームを再起動してみてください

### タッチが反応しない

- 動物の顔の部分をタッチしてください
- 穴ではなく、出てきた動物をタッチ

### 音が出ない

- システムの音量を確認してください
