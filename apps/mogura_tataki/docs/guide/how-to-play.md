# もぐらたたき の遊び方

## 基本的な遊び方

### 1. ゲームを起動

ランチャーから「Mogura」を選択、または単体で起動：

```bash
python apps/mogura_tataki/main.py
```

### 2. 画面の見方

```
┌──────────────────────────────────────────────┐
│ [←]        もぐらたたき                       │
├──────────────────────────────────────────────┤
│                                              │
│     ⭕        ⭕        ⭕                   │
│    (穴)      (穴)      (穴)                  │
│                                              │
│     ⭕        ⭕        ⭕                   │
│    (穴)      (穴)      (穴)                  │
│                                              │
│         どうぶつをタッチしよう！               │
└──────────────────────────────────────────────┘
```

### 3. 動物が出てくる

穴からかわいい動物が顔を出します：

- 🐰 うさぎ（ピンク）
- 🐻 くま（オレンジ）
- 🐱 ねこ（きいろ）
- 🐶 いぬ（みずいろ）
- 🐤 ひよこ（きいろ）
- 🐸 かえる（みどり）

### 4. タッチする

動物が出てきたら、顔をタッチ！

- タッチすると **音が鳴る**
- **キラキラ** が飛び出す
- 動物は **引っ込む**

### 5. 続けて遊ぶ

次々と動物が出てくるので、見つけたらタッチしましょう！

## 操作方法

| 操作 | 動作 |
|------|------|
| 動物をタッチ | 動物が引っ込む＋エフェクト |
| 戻るボタン [←] | ランチャーに戻る |
| ESC キー | ランチャーに戻る |

## ゲームの特徴

### ゆっくりペース

- 動物は **2〜4秒** 出ています
- 焦らなくても大丈夫！

### スコアなし

- 点数やタイマーはありません
- 自由に遊べます

### タッチしなくてもOK

- タッチしなくても動物は自然に引っ込みます
- 失敗という概念がありません

## お子さまへの声かけ例

- 「あ！うさぎさんだ！タッチできるかな？」
- 「くまさんが出てきたね〜」
- 「すごい！タッチできたね！」
- 「次はどこから出てくるかな〜？」

## トラブルシューティング

### 動物が出てこない

- しばらく待ってみてください（最初は1秒後に出現）
- ゲームを再起動してみてください

### タッチが反応しない

- 動物の顔の部分をタッチしてください
- 穴ではなく、出てきた動物をタッチ

### 音が出ない

- システムの音量を確認してください
