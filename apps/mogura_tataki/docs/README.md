# Mogura Tataki - もぐらたたき

穴から出てくる動物をタッチするゲーム。1〜2歳児向けに、反応速度と動物の認識を育みます。

## 概要

- **対象年齢**: 1〜2歳
- **目的**: 反応速度・動物の認識・目と手の協調
- **操作**: タップのみ
- **ゲーム時間**: 30秒

## ゲームフロー

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│  スタート画面   │ ──▶ │   プレイ画面    │ ──▶ │   結果画面      │
│                 │     │                 │     │                 │
│ [ゲームスタート]│     │ タイマー: 30秒  │     │ 15 てん         │
│                 │     │ スコア表示      │     │ すごーい！      │
│                 │     │                 │     │ [もういちど]    │
└─────────────────┘     └─────────────────┘     └─────────────────┘
```

## 特徴

- **30秒チャレンジ**: 赤ちゃんの集中力に合わせた短いゲーム時間
- **スコア表示**: タッチした数をカウント
- **褒め言葉**: 結果画面でポジティブなフィードバック
- **ゆっくりペース**: 幼児でも追いつける速度
- **楽しいフィードバック**: タッチでパーティクルと音
- **かわいい動物**: 親しみやすいキャラクター

```bash
python apps/mogura_tataki/main.py
```

## 技術仕様

- グリッド: 3列 × 2行 = 6つの穴
- ゲーム時間: 30秒
- 出現間隔: 1.5秒ごと
- 表示時間: 2〜4秒
- 描画: Pygame プリミティブ（カスタム画像対応）
- 音声: 手続き的生成（カスタム音声対応）
