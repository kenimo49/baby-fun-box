# のりものビュンビュン の遊び方

## 基本的な遊び方

### 1. ゲームを起動

ランチャーから「Vehicle Go」を選択、または単体で起動：

```bash
python apps/vehicle_go/main.py
```

### 2. 画面の見方

```

│ [←] のりものビュンビュン                      │ ← タイトルと戻るボタン
├──────────────────────────────────────────────┤

│   ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐        │

│                                              │
```

### 3. 乗り物をタップ

好きな乗り物ボタンをタップすると：

1. **エンジン音が鳴る** - 乗り物ごとに異なる音
2. **乗り物が走り出す** - 画面下部を横切る
3. **エフェクト表示** - 排気ガスや水しぶき

### 4. 走行が終わったら

乗り物が画面外に出ると、自動的に待機状態に戻ります。
好きな乗り物を何度でもタップして遊べます。

## 乗り物の特徴

### 地上の乗り物

| 乗り物 | 特徴 | 音 |
|--------|------|-----|
| くるま | シンプルな乗用車 | ブーン（エンジン音） |
| バス | 大きな車体 | ブーン（低音エンジン） |
| でんしゃ | 3両編成 | ガタンゴトン（走行音） |
| しょうぼうしゃ | 赤い車体とはしご | ピーポー（サイレン） |
| きゅうきゅうしゃ | 白い車体と赤いライン | ピーポー（サイレン） |
| バイク | 小さくて速い | ブルルン（高音エンジン） |

### 空と海の乗り物

| 乗り物 | 特徴 | 音 |
|--------|------|-----|
| ひこうき | 斜め上に飛行 | ゴーッ（ジェット音） |
| ふね | 波に揺られて進む | ボーッ（汽笛） |

## 操作方法

| 操作 | 動作 |
|------|------|
| 乗り物ボタンをタップ | 乗り物が走り出す |
| 戻るボタン [←] | ランチャーに戻る |
| ESC キー | ランチャーに戻る |

## お子さまへの声かけ例

遊んでいるときに声をかけてあげると、より楽しめます：

- 「あ！くるまだね〜。ブーンって走ったね！」
- 「しょうぼうしゃ、ピーポーピーポーだね」
- 「ひこうきさん、お空に飛んでいったね〜」
- 「ふねがゆらゆら〜、お水の上を進んでるね」

## トラブルシューティング

### 音が出ない

- システムの音量を確認してください
- Pygame のミキサーが初期化されているか確認

### 乗り物が動かない

- 別の乗り物が走行中は新しい乗り物は走り出しません
- 走行が完了するまでお待ちください

### 画面が真っ白

- ゲームの初期化に時間がかかっている可能性があります
- しばらく待つか、再起動してください
