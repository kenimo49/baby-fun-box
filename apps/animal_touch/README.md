# Animal Touch - どうぶつタッチ

## 概要

画面に表示された動物をタップすると、鳴き声が聞こえてアニメーションする、1〜2歳児向けの教育的なゲームです。

## 対象年齢

**1〜2歳**

### この年齢に適している理由

- **大きなタッチターゲット**: 画面中央に大きく表示される動物
- **教育的要素**: 動物の名前と鳴き声を学べる
- **即座のフィードバック**: タップするとジャンプアニメーション + 鳴き声
- **シンプルな操作**: タップのみ
- **自動切り替え**: 8秒で次の動物に自動で切り替わる

## 登場する動物

| 動物 | 鳴き声 |
|------|--------|
| いぬ | ワンワン！ |
| ねこ | ニャー！ |
| うし | モー！ |
| ぶた | ブーブー！ |
| ひつじ | メェー！ |
| にわとり | コケコッコー！ |
| カエル | ケロケロ！ |
| ライオン | ガオー！ |

## 遊び方

1. 画面中央に動物が表示されます
2. 動物をタップすると鳴き声が聞こえます
3. 矢印ボタンで前後の動物に切り替えられます
4. 8秒経つと自動的に次の動物に切り替わります

## 操作方法

| 操作 | アクション |
|------|----------|
| 動物をタップ | 鳴き声を聞く |
| 左矢印 / ← キー | 前の動物へ |
| 右矢印 / → キー | 次の動物へ |
| スペースキー | 鳴き声を聞く |
| 戻るボタン / ESC | ランチャーに戻る |

## 実行方法

### 統合アプリ経由

```bash
python main.py
# ランチャーでAnimal Touchを選択
```

### 単体実行（開発用）

```bash
python apps/animal_touch/main.py
```

## 技術的な詳細

### 特徴

- **プロシージャル描画**: 外部画像ファイル不要で動物を図形で描画
- **プロシージャル効果音**: 周波数を変えて各動物の鳴き声を生成
- **バウンスアニメーション**: タップ時に動物がジャンプする
- **自動切り替え機能**: 8秒で次の動物に自動遷移
